Available Database Views

Students can query these views:

  • view_devices - All sensor devices (id, name, location, exposure, status, created_at)
  • view_sensors - All sensors with their device info (sensor_id, device_id, sensor_type, unit)
  • view_measurements - All measurements with device and sensor info (value, timestamp, device_name, sensor_type)
  • view_latest_measurements - Most recent measurement per sensor
Example Tasks

Beginner:

  • "List all devices in the database" (SELECT * FROM view_devices)
  • "Find devices with status 'active'" (WHERE clause)

Intermediate:

  • "Find average temperature by device" (GROUP BY, AVG)
  • "Show measurements from the last 24 hours" (Date filtering)
